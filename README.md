# シフトスケジューリング最適化（Shift Scheduling Optimizer）

N人のスタッフをM日間・3シフトに割り当てるシフトスケジューリング問題を、MIP・CP-SAT・ヒューリスティクスの3手法で比較実装したポートフォリオです。

👉 **同一問題インスタンスに対して3手法を実行し、解の品質・求解時間を定量的に比較できます。**

## 解決できる課題

- 手動シフト作成では制約を同時に満たすのが難しい
- 複数の最適化手法の性能差を実際に比較したい
- 最適性保証のある手法と高速な近似手法の使い分けを理解したい

## 問題設定

**N人のスタッフをM日間・3シフト（早番・遅番・夜勤）に割り当てる**

### ハード制約（必ず満たす）

| 制約 | 内容 |
|------|------|
| 必要人数確保 | 各日・各シフトに指定した人数を必ず配置 |
| 1人1日1シフト | 同一スタッフが同じ日に複数シフトに入らない |
| 連続夜勤制限 | 3日連続夜勤を禁止 |

### ソフト制約（違反するとペナルティ）

| 制約 | 内容 |
|------|------|
| 週の労働日数上限 | 週の勤務日数が上限を超えたらペナルティ |
| 希望休の考慮 | 希望休の日に勤務したらペナルティ |

---

## 実装アルゴリズム

### MIP（混合整数計画法）

PuLP + HiGHS を使用。シフト割り当てを0-1変数として定式化し、分枝限定法で最適解を求めます。

**決定変数:**

$$x_{n,d,s} \in \{0, 1\}$$

スタッフ $n$ が日 $d$ のシフト $s$ に入るとき1。

**目的関数:**

$$\min \quad \alpha \sum_{n,w} \text{over}_{n,w} + \beta \sum_{(n,d) \in \text{requests}} \text{violation}_{n,d}$$

ソフト制約違反のペナルティを最小化します。

### CP-SAT（制約プログラミング + SAT）

OR-Tools の CP-SAT ソルバーを使用。CDCLベースのSATソルバー・制約伝播・LP緩和を組み合わせた手法で、スケジューリング問題に特に強みを持ちます。

### ヒューリスティクス（貪欲法 + 局所探索）

外部ソルバー不要のスクラッチ実装。優先度ルールによる初期解構築と、差分計算・近傍サンプリングによる高速な局所探索を組み合わせています。

## ベンチマーク結果

| スタッフ数・日数 | MIP | CP-SAT | ヒューリスティクス |
|----------------|-----|--------|-----------------|
| 8人・14日 | 0.10秒 | 0.12秒 | 0.005秒 |
| 20人・30日 | 0.36秒 | 0.21秒 | 0.009秒 |
| 50人・60日 | 1.76秒 | 0.98秒 | 0.031秒 |
| 100人・90日 | 5.26秒 | 4.03秒 | 0.121秒 |

MIP・CP-SATは最適性保証あり、ヒューリスティクスは最適性保証なし（高速な近似解）。

## アルゴリズムの使い分け

| 観点 | MIP | CP-SAT | ヒューリスティクス |
|------|-----|--------|-----------------|
| 最適性保証 | あり | あり | なし |
| スケジューリング適性 | 良好 | 非常に得意 | 実装次第 |
| 大規模問題 | 計算時間増大 | 比較的安定 | 高速 |
| 実装の透明性 | ソルバー任せ | ソルバー任せ | ロジックが見える |

## 主な機能

- **パラメータの柔軟な設定**: スタッフ数・日数・シフト別必要人数・ペナルティ係数をUIで変更可能
- **3手法の個別実行**: 各手法ページで個別に実行・結果確認が可能
- **一括比較**: 比較ページで未実行の手法を自動実行し、結果を横並びで比較
- **シフト表のカラー表示**: 早番・遅番・夜勤を色分けして視覚的に確認
- **勤務サマリー**: スタッフ別の早番・遅番・夜勤・合計勤務日数を表示
- **求解時間・目的関数値の比較グラフ**: Plotlyによるインタラクティブなグラフ

## 技術的なポイント

- **3手法の統一インターフェース**: `ShiftProblem`・`ShiftSolution` で問題と解を共通フォーマット化し、同一インスタンスでの比較を実現
- **ソフト制約の線形化**: 週の超過勤務・希望休違反を補助変数で線形化し、MIP・CP-SATともに目的関数に組み込む
- **CP-SATの整数スケーリング**: CP-SATは整数演算のみ対応のため、浮動小数点ペナルティ係数を整数にスケーリングして対応
- **差分計算による局所探索の高速化**: ヒューリスティクスの局所探索で全体再評価を避け、交換箇所のみの差分計算でO(1)に近い評価を実現
- **近傍サンプリング**: 全スタッフペアの総当たりを避け、ランダムサンプリングで大規模問題にも対応
